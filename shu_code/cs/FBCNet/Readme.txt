Users need to run ./code/get_data.py to prepare data for FBCNet toolbox. In get_data.py you can shoose different test session from session 2 to session 5, where session 1 is the training session.

Then run ./code/classify/ho.py. 
Before running ho.py, you need to configure two parameters. The first parameter represents the dataset, and input 2 represents the SHU Dataset. The second parameter represents the algorithm, you can choose FBCNet, eegNet, or deepConvNet.
